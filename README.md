# Gulp 4 Mini Starter Kit v 1.0 (alpha)

---

- ##### Automatic graphics conversion
- ##### Compressing your code | HTML | Css | Sass | Java Script |
- ##### Es5 support with babel
- ##### Auto reload page on changes
- ##### Compressing your Java Script code with Webpack 5

---
#### Install dependencies

      npm i

###### Development mode

      npm start || gulp

###### Production mode

      npm run build


### Plugins
      - Html -                            
         gulp-htmlmin
      
      - Cs & Preprocessors -
         Autoprefixer
         Concat & Csso
         Sass

      - Java Script -      
         Webpack
         Webpack-stream
         Babel / Preset-env

      - Graphics -         
         Imagemin

      - Other -
         Browser-sync
         Size
         Del
