# Gulp 4 Mini Starter Kit v 1.0 (alpha)

---

- ##### Automatic graphics conversion to webp and optimization
- ##### Compressing your code | HTML | Css | Sass | Java Script |
- ##### Es5 support with babel
- ##### Convert Fonts to woff / woff2
- ##### Auto reload page on changes
- ##### Compressing your Java Script code with Webpack 5

---

#### Install dependencies

      npm i

###### Development mode

      npm start || gulp

###### Production mode

      npm run build

### Plugins

      - Html -
         htmlmin
         webpHTML

      - Css & Preprocessors -
         Autoprefixer
         gulp-shorthand
         Concat & Csso
         Sass

      - Fonts
         Fonter
         ttf2woff2

      - Java Script -
         Webpack
         Webpack-stream
         Babel / Preset-env

      - Graphics -
         Imagemin
         Webp

      - Other -
         Browser-sync
         newer
         Size
         Del
